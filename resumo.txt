////////////////////////////////////////////////////////////////////

Arquivo .env contém o a minha URL e senha do MongoDB
Mongoose serve para validar e garantir que os meus dados vão ser enviados
Mongoose retorna uma Promise

Sessões servem para salvar dados no navegador do úsuario(ala facebook)
MongoStore serve para salvar as sessões no servidor da base de dados, e não na memória
Flash messages são mensagens autodestrutivas de sucesso, erro e etc...

Rotas são as rotas da aplicação "/contato" é uma rota para o contato da página

CSRF faz a segurança dos nossos forms, fazendo com que nenhum aplicativo use do nosso POST 

Middleware são funções executadas no meio das Rotas

Arquivos estáticos são acessados diretamentes

Views são os arquivos que renderizamos na tela, como HTML's e etc...

View engine é a engine que estamos usando para renderizar HTML(.ejs)
